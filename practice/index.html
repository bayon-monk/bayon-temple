The One-Minute E-Check

For any significant action, ask:

  1. N: Does this strengthen connection? Build trust? Enable reciprocity?
  2. S: Is this true? Coherent? Useful? Does it enable further creation?
  3. C: What does this cost? What friction? What externalities?

If E increases → proceed. If E decreases → reconsider.
