# E-Score Assessment Skill

Assess any topic, scenario, or decision using the E-Score ethical framework.

## Trigger Phrases

- "assess this"
- "what's the e-score"
- "evaluate ethically"
- "N S C analysis"
- "ethical weight"
- @mention with a question or scenario

## The Framework

```
E = (N Ã— S) / C
```

| Dimension | Measures | Scale |
|-----------|----------|-------|
| **N** (Connection) | How many beings affected? How deeply? | 1-10 |
| **S** (Signal) | Does this increase clarity, understanding, truth? | 1-10 |
| **C** (Cost) | What is lost? Autonomy, privacy, resources, trust? | 1-10 |

1. **Identify the scenario** â€” What specific action or decision is being evaluated?
2. **Consider N** â€” Who is affected? How many? How deeply does it touch them?
3. **Consider S** â€” Does this action increase signal (clarity, truth) or noise?
4. **Consider C** â€” What are the costs? To whom? Are they reversible?
5. **Calculate E** â€” (N Ã— S) / C

## Score Interpretation

| E Score | Interpretation |
|---------|----------------|
| 15+ | High ethical weight â€” significant positive impact |
| 10-15 | Moderate positive â€” benefits outweigh costs |
| 5-10 | Balanced â€” consider context carefully |
| 2-5 | Concerning â€” costs may outweigh benefits |
| <2 | Low ethical weight â€” high cost, low benefit |

**Important:** These are guidelines, not verdicts. Context matters enormously.

## Handling Requests

**If asked to assess:**
Provide a full E-Score assessment using the format above.

**If asked to compare:**
Show how the assessment relates to existing scores.

**If challenged:**
Acknowledge the challenge. The framework is a tool, not truth. Ask what framework they'd prefer.

**If the scenario is unclear:**
Ask clarifying questions before assessing. Don't assess vague scenarios confidently.
