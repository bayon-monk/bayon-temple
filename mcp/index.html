MCP Server

E-Score tools for Claude Code & Claude Desktop

The Bayon MCP Server brings the E-Equation framework directly into your AI conversations. Calculate scores, submit scenarios, and browse the Exchange — all without leaving your workflow.

Quick Install

Claude Code:

claude mcp add bayon -- npx -y @bayon_monk/mcp-server

Claude Desktop:

Add to your claude_desktop_config.json:

{ "mcpServers": { "bayon": { "command": "npx", "args": ["-y", "@bayon_monk/mcp-server"] } } }

Available Tools

calculate_e_score

Calculate E-scores inline. Pass N, S, C values and get instant interpretation.

submit_scenario

Submit scenarios to the E-Score Exchange for community rating.

get_feed

Get the live feed of recent scenarios and their proposed scores.

get_entity_score

Look up aggregated E-scores for companies, people, or policies.

Example Usage

"What's the E-score if a company open-sources their AI model? N=8, S=7, C=3."

Claude will use calculate_e_score and respond:

E = (8 × 7) / 3 = 18.67 (High-E)
Net positive contribution to the field.

The E-Equation

E = (N × S) / C

N = Connection (0-10)  — Network density, relationships
S = Signal (0-10)      — Truth value, coherence, generativity
C = Cost (0.1-10)      — Thermodynamic tax, friction, entropy
